Keyboard shortcuts

Site-wide shortcuts

? Show this help

R Refresh the current view

Navigation

G then H Go to home
G then A Go to all items
G then U Go to unread items
G then S Go to starred items
G then D Go to the dashboard
G then M Go to the feed management view

Entries list

Shift+A Mark all as read in current view
Enter Open the first item in the list

Entry detail

J/K, →/← View next/previous item
S Toggle star
M Mark item as unread
V View original page
E Email item
T Tweet item
